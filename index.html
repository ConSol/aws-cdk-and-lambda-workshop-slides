Prerequisites

General
  • AWS account
    (alternatively I can create access keys and email them)
  • AWS CLI: Windows, Linux, Mac
  • AWS access key
    (for AWS CLI)
  • git
  • Visual Studio Code
    or any other IDE or text editor with Typescript support

Prerequisites

Linux
  • Latest version of Node.js
    hint: use nvm
  • AWS CDK: 
    npm install -g aws-cdk@1.12.0

Prerequisites

Other Platforms

Suggested workaround:

  1. Launch an EC2 instance with all the necessary tooling and SSH into it
  2. Edit code locally in your favorite IDE
  3. Commit and push changes
  4. From within the EC2 instance:
    1. pull the changes
    2. run CDK commands
